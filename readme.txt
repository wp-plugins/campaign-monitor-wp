== Description ==

= Easy Opt-ins For Campaign Monitor WordPress Plugin =

The **Easy Opt-ins For Campaign Monitor** WordPress Plugin makes it easy to create highly converting Campaign Monitor sign-up and opt-in forms in less than two minutes.


This plugin is similar to our sister plugin, [Easy Opt-ins For Mailchimp](https://wordpress.org/plugins/mailchimp-wp/), but built for Campaign Monitor. You can find a video & live demo here:
[youtube http://www.youtube.com/watch?v=oDrN-R1voxs]

*   [Find a live demo here.](http://demo.fatcatapps.com/easyoptins/)

= Campaign Monitor Opt-in Forms =

*   Easy Opt-ins For Campaign Monitor implements conversion optimization best practices and guides you through the process of creating a Campaign Monitor subscription form that converts.

*   After building your signup form you can publish it as a widget in your sidebar.


= Features =

*   Opt-in form works with responsive sites
*   Intuitive User Interface
*   Built-in Conversion Rate Optimization Best Practices
*	Customize your sign-up form: change copy, font-size, colors and more.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In your sidebar, select 'Opt In Forms -> Add New' to create a new table
